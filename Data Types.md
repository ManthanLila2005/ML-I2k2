Dictionary function
keys() - get all keys
values()- get all values
item() - get all pairs

get() - fetches data without error if data missing


set functions
add() - adding element to set
remove() - remove a element from set
discard() - remove a element from set but doesn give error if value not found

a|b- union of a \& b
a\&b- intersection of a \& b
a-b - values in first set after removing intersection
a^b - union - intersection


list function

append() - add element to list

extend() - add multiple elements

insert(index,value)- add at a position
remove()- remove a particular value
pop()- remove by index

clear() - remove everything
index() - find index of a value
count() - count occurrence of a value
sort() - sort list
sort(reverse=true) - sort in descending

reverse() - reverse list

copy() - copy a list

len() - find no of elements in list
max()/min() - find minimum or maximum element

sum()- find sum of all element

sorted() - returns a sorted copy
